Sigmoid star in CGC domain

Compute the Sigmoid function, defined as 1/(1 + exp(-r*input)), where r is the learning rate. The Sigmoid function is commonly used in neural networks.

States

learning_rateFloatState The learning rate must be positive. Once the network is well-trained, the learning rate is fixed. If you scale the weights of the neurons in a trained neural network, then you multiply the learning rate by the same factor. Scaling is important, for example, to guarantee that all of the intermediate computations remain in the range [-1,1). "0.5"

Details

A common model of an artificial neuron is a weighted sum of inputs followed by an activation function. The activation function, which is often nonlinear, transforms the net input to a neuron into its activation (level of activity) [1]. The Sigmoid function is one of the most commonly used activation functions in artificial neural networks.

The input to the Sigmoid function can be any number, positive or negative, and outputs a number from 0 and 1. Hence, the Sigmoid function does not suffer from numerical overflow.
