Minor Changes in Ptolemy 0.7.2

  • Support for compiling Ptolemy under Solaris 2.7
  • Support for gcc-2.95.1
  • Support for Cygwin b20.1
  • See the release notes at $PTOLEMY/doc/html/release.html for other minor changes

    • Features present in Ptolemy 0.7.1 that were removed in 0.7.2

  • The CG56, C50, VHDLB and VHDL domains have been moved to an older CG domains palette. The domains have not been thouroughly tested.
  • The Mathematica functionality has not been tested, but it is still present in the sources.
