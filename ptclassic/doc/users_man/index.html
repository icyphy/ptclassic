Ptolemy 0.6 User's Manual

This directory contains the Ptolemy 0.6 User's Manual.

The contents of this directory are contained in ../users_man0.6.tar.gz

The User's Manual is also available as html at ../www/users_man0.6. That directory is available as a tar file at ../users_man0.6.html.tar.gz.

Contents of this directory

print_users_manual
A script that will print the postscript in this directory in correct order (back to front). The script assumes that your PRINTER environment variable is set to the name of the printer you wish to use.
