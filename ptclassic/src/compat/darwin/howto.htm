Ptolemy Classic v0.7.1 on Mac OS X

Below is part of David Evans page at http://www2.cs.pitt.edu/~devans/
This is my first pass at porting Ptolemy Classic to Darwin. There will be changes made and noted here. The plan is to get it up and running first, and then vector optimize the star/pigiRpc code to enhance performance where possible.

Installing Ptolemy requires the following non-standard additions/changes to your system: I take no responsibility for your actions...USE AT OWN RISK!
  • The Ptolemy Classic v0.7.1 source tree available from Berkeley
  • Modified g++ library (or libnqg++.a for not-quite g++...This involves compiling each .cc file separately and removing those that don't compile. - hopefully we can change this with ARCH flags.)
  • libdl or dlcompat from the fink project or apex's ports
  • Create a minimal minmax.h file ... an example is here.
  • Create a minimal a.out.h file ... an example is here.
  • This is our config-Darwin.mk, it changes often, and it may change again in the near future. This file will take care of MOST of the quirks. config-Darwin.mk
    •  The above are the minimum set of additions for building ptolemy. Of course there are other caveats, but almost all should be taken care of by the config-Darwin.mk file.
