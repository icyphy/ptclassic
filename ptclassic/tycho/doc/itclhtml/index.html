Itcl HTML documentation

This directory usually contains the Itcl HTML documentation which is shipped with Itcl2.2 in itcl2.2/html.

If you downloaded Ptolemy with the other.src tar file, then you can make a link in this directory to the sources in $PTOLEMY/src/tcltk/itcl2.2/html: 

cd $PTOLEMY/tycho/doc/itclhtml
rm -f index.html
ln -s $PTOLEMY/src/tcltk/itcl2.2/html/* .

If you did not download the other.src, you can grab the Itcl distribution off the www.tcltk.com website and create the links as above.
