Troubleshooting Tycho

Below is a list of common problems that people have with Tycho. You may also wish to check out Tycho 0.1 Limitations and Bugs and Tycho 0.1 Release Notes.

Problems Starting Tycho

Error Message about "error while autoloading tk_setPalette"

If you see a message like: 
error while autoloading "tk_setPalette": bad option "join": should be
atime, dirname, executable, exists, extension, isdirectory,
isfile, lstat, mtime, owned, readable, readlink, root, size,
stat, tail, type, or writable
	
    invoked from within
"tk_setPalette activeBackground #e6ceb1 activeForeground black \
background #ffe4c4 disabledForeground #b0b0b0 foreground black \
highlightBackground ..."
the you are probably running itcl2.0, but the tclIndex files were prepared with itcl2.1. The workaround is to regenerate the tclIndex files by doing: 
cd $TYCHO
make sources
