Itcl Class Diagrams

Tycho can create Itcl class diagrams and display them with the Directed Acyclic Graph Editor.

tychoClasses.dag

The file $TYCHO/doc/tychoClasses.dag contains a Directed Acyclic Graph of the Tycho classes. This file is generated by running make all or make sources while in $TYCHO/doc.

Itcl Class Lists

The Itcl Class List Editor can be used to generate class diagrams of user supplied classes.

$TYCHO/kernel/tychoMakeClassDAG.tcl

The file $TYCHO/kernel/tychoMakeClassDAG.tcl contains the Itcl procs that are used to create the class DAGs.
