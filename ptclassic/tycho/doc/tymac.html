    • Tycho/Macintosh bugs

    Note that tycho0.1.1 does not work on the Mac. We are making Mac Itcl binaries available so that developers can work on the port.
  • Starting Tycho may crash your mac. The bug seems to be caused by the lines in Tycho.tcl that create the TclShell.
  • There are pathname problems with the FileBrowser. Currently, it is not possible to open a file.
  • Starting tycho requires sourcing the tycho-mac script by hand. There must be a better way.
  • Tycho on the mac should use an installer.

    • Installing Tycho on the Macintosh

    1. The Tycho home page at has a link to the downloads page that contains Tycho sources and Itcl binaries for the Mac.
      Alternatively, you can pick up prebuilt Itcl2.2 Macintosh PowerPC binaries at ftp://ftp.tcltk.com/pub/itcl/itcl2.2-ppc.sit.hqx
    2. Download both the tycho distribution and the Itcl distribution and place them in directories next to each other. For example,
    3. The tycho distribution could be in mac:devel:tycho0.1.1 The itcl distribution could be in mac:devel:itcl2.2-ppc
    4. Move the tycho-mac script from tycho0.1.1:lib:mac to the top level itcl directory. In this example, tycho-mac would end up at mac:devel:itcl2.2-ppc:tycho.
    5. To start up tycho, start Itkwish, then type source tycho-mac.
