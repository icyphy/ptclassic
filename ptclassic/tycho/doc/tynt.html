Tycho under Windows NT

Places to go

  • Tcl/Tk on Windows FAQ
    • Itcl2.2 should be out shortly, and have binaries for NT. As of 12/1/96, we are still using Itcl2.1.

    Compiling itcl2.1

    You can pick up prebuilt Itcl2.1 NT binaries at ftp://mirror.neosoft.com/pub/tcl/NEW/itcl2.1p.NT.tar.Z
  • I had to set the INCLUDE and LIB environment variables by using the system control panel.
  • Before running nmake -f makefile.vc install, be sure that c:\bin is in your path, or the iwidgets installation will fail.
  • In iwidgets2.1.0, I substituted itclsh21 for tclsh75

    • Changes to Tycho

    1. In the default NT setup, exec rm does not work, there is no rm binary. Ditto with exec mkdir. If you download the Cygnus GNU-win32 tools, then you will have mkdir and rm.
    2. To create a tycho script, I created a script called ty.itcl, with the contents: set env(TYCHO) c:/users/cxh/tycho0.1.1 source $env(TYCHO)/kernel then I clicked on the ty.itcl icon, and when I was prompted to select an application, I chose c:/bin/itkwish2.1
