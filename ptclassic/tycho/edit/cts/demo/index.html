Continuous Time Simulation Demos

Author:Jie Liu

General Guide

  • To run the simulation, click on the tycho code below. The GUI will be loaded.
  • Click on run in the devel menu.
  • Wait for a few second to allow creating the socket connection.
    • If there's a message like: Socket Error, which means the socket has already been allocated for other resources, then please close the run window and try again.
  • Adjust start time, stop time, and step size.
  • Llick start to start the simulation
  • During the simulaiton, the slider, if there is any in the run window, can be used to adjust the parameters on the fly.
  • Pause, Resume, and Stop are used for execution control.

Demos

1. MEMS: A sonator with a comb finger sensor. (Coyote Systems) 

 ::tycho::openContext Coyote2.cts

  • After bring up the run window, enter 15 in the stop time box.
  • Click Start.

2. Lorenz System:A simple system with chaotic behavior 

 ::tycho::openContext Lorenz.cts

  • After bring up the run window, enter 50 in the stop time box.
  • Click Start.
