Installation


If you are running the Slate from within Tycho, then you shouldn't need to do anything more. Otherwise, read on. We assume that you already have [incr Tcl] 3.0 or Tcl 8.0 installed and know how to run it.

Note that the Slate has been tested only under Solaris and Windows NT. It should work as advertised on other platforms, but we haven't tried it.

  • Download
  • Install
  • Check

Download

If you are running [incr Tcl] 3.0, you only need the slate package itself. If you are running Tcl 8.0, you also need to get the obstcl package.

slate
This is the slate package itself. It contains everything you need if you are running [incr Tcl] 3.0.
  • slate.zip
  • slate.tar.gz

obstcl
obstcl is a simple Tcl-only object-oriented layer for Tcl. You need this to run the Slate with Tcl 8.0.
  • obstcl.zip
  • obstcl.tar.gz

Install

These instructions assume that you do not already have Tycho installed.

  1. Download the packages you need into a suitable directory. This is either the lib directory of your Tcl or [incr Tcl] installation, or some other directory where you put Tcl packages. To see what the lib directory is, start itclsh or tclsh and type: 
        file dirname [info library]

  2. Change to that directory and do either: 
        unzip -a slate.zip
    unzip -a obstcl.zip

    or 

        gzcat slate.tar.gz | tar -xf -
    gzcat obstcl.tar.gz | tar -xf -

    If you are running under windows and have WinZip installed, you can also double-click on the files.

  3. Make sure that the Tcl package loader will be able to find these packages. If you didn't install them in the lib directory of your Tcl installation, then I suggest that you set an environment variable TCL_PACKAGES to the directory containing the two packages. (This is needed for the demo and test scripts.)

  4. Fix the demo and test startup scripts. The second line of these scripts will say something like 
        exec itkwish3.0 "$0" "$@"
    (Windows users: the following assumes you have bash from Cygwin installed. If not, and you need a DOS batch file, send me email).

    Change the itkwish3.0 to:

    • wish80, to run Tcl 8.0 on Windows
    • wish8.0, to run Tcl 8.0 on Unix
    • itkwish30, to run [incr Tcl] 3.0 on Windows
    • itkwish3.0, to run [incr Tcl] 3.0 on Unix
    or to whatever alias or symbolic link you have for running wish or itkwish.

Check

To verify the installation, start wish 8.0 or itkwish 3.0. If the Slate is not in your Tcl library directory, go: 

  lappend auto_path $env(TCL_PACKAGES)
Then go: 
  package require slate
You should get "3.0". Then try: 
    ::tycho::slate .s
    pack .s
    raise .
    .s create Frame 50 50 100 100 -color green
Any problems, send me email.

Now you can try out the demos or run the tutorial. When you use the Slate in your own programs, just be sure to do 

    package require slate

before trying to create a new Slate.

(Note to Tycho users: in Tycho, the Slate package is called tycho.edit.slate.)
