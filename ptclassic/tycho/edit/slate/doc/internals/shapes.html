Shape classes

The Slate gives every item a shape, which is used to manipulate the item in various ways. Each shape is an "uninstantiable" class, with the root class of the hierarchy being the Shape class. Here is the list of shapes, and for each, the features it supports. Heading are hyperlinked to the class documentation for the appropriate class.

Note: some complex items choose to override the default features given by its shape. Note also that (i) an item can be made to appear rectangular when method such as feature and reshape are called, by supplying the flag -region; and (ii) an item is treated as a rectangle by the region method, which is like coords but queries or manipulates the rectangular region of the Slate covered by the item.

  • point
  • rectangle
  • oval
  • line
  • polygon

point

The item has two coordinates only. Generally, items of "point" shape have a -anchor option that positions them relative to the point. Items with this shape include text, bitmap, image, and Axis.

Features

  • origin: The coordinates of the item.

rectangle

The item has four coordinates, being the left, top, right, and bottom coordinates. Items with this shape include rectangle, Frame, LabeledRect, and LabeledFrame.

Features

  • n: The center of the top edge.
  • s: The center of the bottom edge.
  • e: The center of the right edge.
  • w: The center of the left edge.
  • ne: The top-right corner.
  • nw: The top-left corner.
  • se: The bottom-right corner.
  • sw: The bottom-left corner.
  • c or center: The center of the rectangle.

oval

The item has four coordinates, being the left, top, right, and bottom coordinates. Items with this shape include oval and LabeledOval.

Features

  • hradius: The point at the right side of the oval; changing this feature changes the horizontal width of the oval.
  • vradius: The point at the top side of the oval; changing this feature changes the vertical height of the oval.
  • c or center: The center of the oval.
  • circumference-x-y: The point on the circumference that intersects the vector (x, y) drawn from the center.

line

The item has four or more coordinates, each pair being one vertex of the line. Items with this shape include line and SmartLine.

Features

  • vertex-n: The n'th vertex of the line.
  • start: The start of the line.
  • end: The end of the line.

polygon

The item has four or more coordinates, each pair being one vertex of the polygon. The last pair must be identical to the first pair. Items with this shape include polygon and Solid.

Features

  • vertex-n: The n'th vertex of the polygon.
