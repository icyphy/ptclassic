Writing user-defined items

You can create your own item types in Tcl by subclassing the ComplexItem class. The tutorial gave a simple example of how to write such a class. These links point to more detailed information on writing ComplexItem subclasses.

Of course, if you get stuck, looking at the classes supplied with the Slate might help too.
