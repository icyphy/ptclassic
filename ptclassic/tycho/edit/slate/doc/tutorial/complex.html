Complex items

The slate has kinds of item: simple items, which are those supplied by the Tk canvas, and complex items, which are items composed out of other items. The ComplexItem item type is the generic complex item type, and can be used to dynamically create arbitrary items.

For example, we can create a new complex item by doing this: 

  package require slate.demo
  set slate [::demo::newslate "A complex item example"]
  set citem [$slate create ComplexItem 50 50 100 100]
You can't see the item, because it doesn't actually contain anything yet. The coordinates given the region to be occupied by the item so that methods such as coords will operate correctly.

Now let's add some items to it. For the sake of example, let's add a pair of lines and an oval: 

  $slate createchild $citem line 50 50 100 100
  $slate createchild $citem line 50 100 100 50
  $slate createchild $citem oval 60 60 90 90 -fill green
This new item will respond to slate operations just like the other items we have seen so far in this tutorial. For example, we can move it: 
  $slate move $citem 10 10

We can change its coordinates 

  $slate coords $citem 100 100 200 170

To prove a point, let's add some grab-handles to the item and make it moveable, so you can fiddle with it yourself: 

  $slate addtag moveable withtag $citem
  $slate grapple $citem
You can also define your own item types, as described in a later section of this tutorial.
