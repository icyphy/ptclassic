Introduction

The slate, like the Tk canvas, creates graphical items. The items provided by the Tk canvas have lower-case names, like rectangle and line. The user-defined items that the slate supports have capitalized names, like Frame and SmartLine.

For example, let's create a slate and place a Frame item on it (so called because it mimics the Tk frame widget): 

  package require slate.demo
  set slate [::demo::newslate "A simple example"]
  set frame [$slate create Frame 50 50 100 100 \
              -color green -borderwidth 4 -relief ridge]
The arguments to the create method are the item type are its coordinates, followed by options and values. Option names always start with a leading dash.

To see the options of an item, go: 

  $slate itemconfigure $frame

To change the value of an options of an item, go, for example: 

  $slate itemconfigure $frame -relief sunken

To get the value of a single option, go, for example: 

  $slate itemcget $frame -relief

To see the coordinates of an item, go: 

  $slate coords $frame

To change the coordinates of an item, go, for example: 

  $slate coords $frame 80 80 160 120

To move an item, go, for example: 

  $slate move $frame 10 10

To scale an item, go, for example: 

  $slate scale $frame 100 100 2.0 0.5

To get the bounding box of an item, go: 

  $slate bbox $frame

(This is not so useful for rectangular items, as you can just use their coordinates. The bounding box in Tk is not very "tight" -- it tends to be one or two pixels larger than the actual bounds of an item.)

Finally, to delete an item, go: 

  $slate delete $frame
