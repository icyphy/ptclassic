Tycho Text Editors and Shells

The Tycho text editor package, tycho.edit.textedit, provides a variety of specialized text editors and command-line shells. All of these are derived from the Edit class, and therefore inherit all of its features. They add a wealth of featues such as syntax-sensitive indentation and coloring, automatic documentation generation, and source code compilation.

Documentation available:

User's guides
Guides to the individual text editor and shell widgets. These guides are also available from the Help menu within the Tycho GUI.

Class documentation
Documentation for all of the text editor and shell classes, generated from the source code by the tydoc utility.
