MyRamp star in SDF domain

Generate a ramp signal, starting at "value" (default 0) and incrementing by step size "step" (default 1) on each firing.

Derived from: SDFStar
Location: SDF main library
Version: 1.1 "10/13/96"
Author: D. G. Messerschmitt

Outputs

outputfloat

States

stepFloatState Increment from one sample to the next. "1.0"
valueFloatState Initial (or latest) value output by Ramp. "0.0"
