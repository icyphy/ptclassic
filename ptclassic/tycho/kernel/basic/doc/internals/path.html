Pathname manipulation procs

The Tycho kernel contains a series of procs defined in the ::tycho name space. These procs are not associated with any particular class. These procs are defined in $TYCHO/kernel/Path.tcl. The automatically generated documentation for this file can be viewed from the Code Index menu choice under the Help menu.

The pathname functions include functions to expand pathnames, create unique filenames, create temporary filenames, remove files and make directories
