Pathname manipulation procs

The Tycho kernel contains a set of procedures for manipulating pathnames and URLs. The documentation for these procedures can be found in:

  • Path.html
  • URLPath.html

The pathname functions include functions to expand pathnames, create unique filenames, create temporary filenames, remove files and make directories
