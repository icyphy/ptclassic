Tycho Kernel GUI Package Programmer's Guides

Tycho provides a set of procedures and classes that can be used in scripts, procedures, and class definitions created by the user. This set of documents introduces this infrastructure at a descriptive level, suitable for a user intending to write simple scripts. For more serious software development, the class documentation should be referenced. Not all classes are described here. Only the ones that are most useful to a casual script writer are described.
