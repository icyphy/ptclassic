User's Guide to the BuilderDialog Widget

The BuilderDialog widget allows the user to build documentation and indexes on platforms that do not have a make binary.

The BuilderDialog widget consists of a set of radio buttons that allow the user to choose different operations, and a set of regular buttons that invoke different operations.

You can access the BuilderDialog from the Tools submenu under Window toolbar menu.

BuilderDialog Radio Buttons

The BuilderDialog Radio Buttons allow the user to select an operation to be performed when the user hits the OK or Build buttons. Below are the radio buttons.
Full
Do a Full build; takes about 10 minutes The Full build performs all of the operations below.
The Full build takes a long time because it creates the Itcl HTML documentation, then creates the code index from the Itcl HTML documentation, and then recreates the Itcl HTML documentation using the code index.
Glimpse
Build the Glimpse database.
Itcl HTML docs
Build the Itcl HTML docs; takes about 9 minutes
Indices Only
Build only the Tycho Concept and Code Indices
Class Diagram Only
Build only the Class Diagram. See the ItclClassList documentation for the way to build custom class diagrams.

BuilderDialog Buttons

OK
Run the operation that is selected in the radio buttons, then destroy the BuilderDialog window.
Cancel Destroy the BuilderDialog window without running any build operations.
Build
Run the operation that is selected in the radio buttons, but do not destroy the BuilderDialog window.
