User's Guide to the BuildDialog Widget

The BuildDialog widget allows the user to build documentation and indexes on platforms that do not have a make binary.

The BuildDialog widget consists of a set of radio buttons that allow the user to choose different operations, and a set of regular buttons that invoke different operations.

BuildDialog Radio Buttons

The BuildDialog Radio Buttons allow the user to select an operation to be performed when the user hits the OK or Build buttons. Below are the radio buttons
Full
Do a Full build, takes about 10 minutes
Indices Only
Build only the Tycho Concept and Code Indices
Class Diagram Only
Build only the Class Diagram

BuildDialog Buttons

OK
Run the operation that is selected in the radio buttons, then destroy the BuildDialog window.
Cancel Destroy the BuildDialog window without running any build operations.
Build
Run the operation that is selected in the radio buttons, but do not destroy the BuildDialog window.
