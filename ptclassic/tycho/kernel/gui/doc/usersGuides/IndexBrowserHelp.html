User's Guide to the Tycho Index Browser

The index browser is list browser specialized for browsing through lists of index entries in Tycho documentation. An index browser can be opened from most Tycho windows through pull-down help menu. In this case, the index browser opens the Tycho master index. In general, any file with the extension ".idx" will be opened using an index browser. Although the Tycho index browser is designed to be relatively familiar to most users, it has some unique features. Here is a summary of helpful hints:

  • Double clicking on an item will select it and open it.
  • Tab is used for name completion.
  • Repeated Tabs will scroll through matching items.
  • Up and down arrows plus the emacs-like equivalent bindings can be used to navigate through the list of index items (if you are using name completion, use these to navigate through the list of possible completions).
  • The window can be resized to see more of the items (use whatever mechanisms your window manager provides).

See Tycho Documentation for instructions on how to create an index.
