The HTML Class

The HTML class, derived from File, is a top-level window that renders HTML in a text widget. It uses the HTML library written by Stephen Uhler of Sun Microsystems, which supports all the features of HTML 2.0.

Currently, there is currently no support in Tycho for HTTP hyperlink references, so you cannot read documents directly on the internet from within Tycho. If Tycho encounters a HTML reference that it does not understand, then Tycho with ask the user if they want to start up Netscape. (The Macintosh does not support exec, so this will not work on the Mac). We are working on adding HTTP support to Tycho.

A sample document illustrates the HTML tags that are supported.

The HTML class is used for all Tycho documentation. It supports an extension of HTML with embedded Tcl scripts. If the user double clicks on a script, the Tcl script is executed in the global namespace.

You may create an instance of the HTML class as follows: 

::tycho::view HTML -file $TYCHO/doc/introduction.html
There are many public methods associated with this class. See the class documentation for details.

HTML has a number of options. For example: 

set h [::tycho::view HTML \
	-data "<h1>Initial header in the HTML widget</h1>" \
	-textwidth 60 \
	-scrollbar 0 \
	-textheight 20]
By default, an HTML widget is read-only, so you cannot modify its contents. You can change this, and read in a file as follows: 
$h configure -readonly no
$h insertData " Some <b>More</b> html"
Notice that Emacs-like key bindings can be used to navigate through the text. There are a large number of other public methods for HTML. See the class documentation. To delete the above example: 
delete object [$h displayer windowName]
