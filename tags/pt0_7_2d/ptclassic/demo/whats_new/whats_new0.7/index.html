What's New in Ptolemy 0.7

Unlike earlier releases of Ptolemy, Ptolemy 0.7 does not contain a "What's New" facet, instead we are using a new feature in Ptolemy 0.7 which allows us to create icons that contain html files such as this file.
For example, the icon for this file is in the $PTOLEMY/demo/whats_new/init.pal palette.

These page is best viewed with Tycho, as it uses extensions to html.

  • Some of the links lead to Ptolemy facets, which a regular browser won't know how to handle.
  • Some of the links use environment variables, such as $PTOLEMY or $TYCHO to link to local web pages.
