README file for Ptolemy
-----------------------

In order to run Ptolemy, do the following:

% setenv PTOLEMY This_Directory
% set path = ($PTOLEMY/bin $path)
% cd $PTOLEMY/demo
% pigi

Where "This_Directory" is the directory where you installed
Ptolemy.  This README file should be found at $PTOLEMY/README

For further information, see $PTOLEMY/doc/index.html

The Ptolemy home page at http://ptolemy.eecs.berkeley.edu has further
information about Ptolemy.

ptolemy.faq is the Ptolemy Frequently Asked Questions file.
