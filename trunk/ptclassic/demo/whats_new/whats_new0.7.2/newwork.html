New work in Ptolemy 0.7.2

The Adaptive Computing Systems (ACS) Domain - Eric Pauer, John Smith, James Lundblad
The ACS domain uses different styles of stars than the rest of Ptolemy. ACS splits a star into a single interface (called a Corona), and one or more implementations, called Cores.
See the ACS Documentation for more information.
Windows NT Port - Christopher Hylands
Ptolemy 0.7.2 runs under Windows NT using the Cygwin32 compiler and the Cygwin32 X11R6 binaries. To run pigi under NT, you will need a NT X server. For more information, see the Ptolemy under Windows NT page.
