Tcl bugs

  • Under Cygnus GNU-win32, the Tcl core source code must be modified.
  • The port currently does not work with Itcl

    • Build Bugs

  • If you see lines like: 
    common.mk (80) no %
    Then try calling make with 
    make PTLANG=$PTOLEMY/bin.nt4/ptlang
  • While compiling, there are lots of warnings about 
    src/kernel/PortHole.h:221: warning base class `class NamedObjList'
has a non-virtual destructor.
    The fix is to use the -Wno-non-virtual-dtor option.
