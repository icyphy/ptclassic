Weighted_List star in DE domain

An input value is read in. Depending on its value one or more counters in a list are increased. The outputs are the contents of the counters, weighted by the number of all incoming values. Thus, the probability that an incoming value is smaller than a specific value is given out for all values in the list. Note that all outputs have the same timestamp.

Derived from: DEStar
Location: The WiNeS-Library
Version: 1.1 "06/24/97"
Author: J. Voigt

Inputs

CIRfloat CIR-value for one time slot

Outputs

testxfloat outage probabilities
testyfloat outage probabilities
