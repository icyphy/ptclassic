propag_uldl star in DE domain

This star implements a path loss model. A specific enviroment is read in from a AutoCad-file. All obstacles which are in a path are searched for and their penetration loss is added to the path loss caused by free space propagation. The resulting path loss is given out. WARNING: I don't recommend anybody to trace down the following code. It is kind of a quick hack and not very readable for others. Also, please excuse some comments in German I left behind. They are to many to translate them all ... Moreover, the implementation highly depends on the environment I use. This is not a generic path-loss-model for any AutoCad-enviroment !!!
