Tycho File Template

Itcl files written for Tycho should conform to a specific style so that documentation can be automatically generated. A template is provided to be used as a starting point for all itcl files written for Tycho. All files should begin with a standard header. For details regarding this header, see Tycho File Headers.

You can insert the Itcl and Java templates into an Itcl or Java editor by using the File Template menu choice under the Special menu. For more information, see Creating New Class Files
