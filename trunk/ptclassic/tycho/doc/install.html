Tycho Installation Instructions

  1. See http://ptolemy.eecs.berkeley.edu/tycho/Tycho.html for the various distributions that are available.

  2. Go to the appropriate page for your platform:
  3. Tycho under Unix
  4. Tycho on the Macintosh
  5. Tycho under Windows NT
  • When Tycho starts up, it checks for certain documentation files. If these files are not present then a dialog box pops up asking if you want to run the BuilderDialog. The BuilderDialog has menu choices that will create the necessary Tycho documentation.
