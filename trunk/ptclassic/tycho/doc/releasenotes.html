Release Notes

Note that the Ptolemy group releases all software with version numbers "0.n", where the "0" prefix in meant to remind the user that this is not commercial software. It does not mean that we do not have faith in the software.

  • Version 0.1.1 (Released 12/96)
  • Version 0.2 (Released 5/97)
  • Version 0.2.1 (Released 6/98)
  • Version 0.3 (Released 1/99)
