Labeled items

The Slate has a number of item types that extend simpler item with a text label and with arbitrary graphics. All of these item types inherit from the LabeledItem class as well as some other class.

LabeledItem

LabeledItem is the superclass of labelled items.

Options:

-text
The label of the item. If there are graphics also displayed, the label is placed outside the item; if not, the label is displayed outside the item. By default, this option is "", which means that nothing is displayed.
-graphics
The graphics to draw on the item. This option consists of zero or more item specs, where an item spec is the item type, coordinates, and options. The coordinates are specified as a percentage of the item size. Item specs are separates by newlines or semi-colons.
-anchor
The anchor position of the text label relative to the region occupied by the item. The anchor can be any of the compass points -- n, s, e, w, ne, se, nw, sw -- and c or center. If it is c or center and there are graphics drawn on the item, the label is drawn underneath the item (as though the anchor was s).

Bugs: (none)

Enhancements: (none)

LabeledOval

LabeledOval is an oval with a text label and graphics.

Options: LabeledOval has the options of LabeledItem and the following:

  • -fill. The color with which the oval is filled. By default, it is "", which means that the oval is not filled.
  • -outline. The color of the outline of the oval. If "", the oval has no outline. By default, it is black.
  • -width. The width of the outline of the oval, in pixels. By default, it is one.
In all other respects, it can be treated like any other oval item.

Enhancements: (none)

Example: Create a labelled oval: 

  source $tycho/editors/slate/doc/internals/newslate.itcl
  $slate create LabeledOval 40 40 100 80 -fill white -outline blue \
		-width 4 -text Foo -tags moveable

LabeledRect

LabeledRect is a rectangle with a text label and graphics.

Options: LabeledRect has the options of LabeledItem and the following:

  • -fill. The color with which the oval is filled. By default, it is "", which means that the rectangle is not filled.
  • -outline. The color of the outline of the rectangle. If "", the rectangle has no outline. By default, it is black.
  • -width. The width of the outline of the rectangle, in pixels. By default, it is one.
In all other respects, it can be treated like any other rectangle item.

Enhancements: (none)

Example: Create a labelled rectangle: 

  source $tycho/editors/slate/doc/internals/newslate.itcl
  $slate create LabeledRect 40 40 100 80 -fill blue -outline red \
		-anchor nw -text Foo -tags moveable \
		-graphics "line 0 0 100 100; line 0 100 100 0"

LabeledFrame

LabeledFrame is a Frame with a text label and graphics. It has all the options of LabeledItem and all the options of Frame. In all other respects, it can be treated like any other Frame item.

Enhancements: (none)

Example: Create a labelled Frame: 

  source $tycho/editors/slate/doc/internals/newslate.itcl
  $slate create LabeledFrame 40 40 100 80 -color green \
		-text Foo -tags moveable \
		-graphics "Frame 25 25 75 75 -color green -relief sunken"
