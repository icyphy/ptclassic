Features

Every item on slate has a set of features -- that is, named points on the item. The features of an item depends on its shape. Let's create a few items: 

  package require slate.demo
  set slate [::demo::newslate "Interactor examples"]
  set frame [$slate create Frame 120 120 180 170 -borderwidth 4]
  set line  [$slate create line 30 30 30 70 70 70 70 110 -arrow last]
The feature method will return the list of features of an item: 
  $slate feature $frame
In the case of lines (and polygons), the feature set depends on the number of vertexes: 
  $slate feature $line
You can also query an item for individual features: 
  $slate feature $line start end
You can reshape an item by moving one of more of its features. For example, to change the shape of the frame: 
  $slate reshape $frame 10 10 se
For example, to change the shape of the line: 
  $slate reshape $line 10 10 vertex-2 vertex-3
(Lines understand "vertex-0" instead of "start" and "vertex-m" instead of "end," where m is the number of vertexes less one.) You can also move an item so that one of its features lies at a certain point: 
  $slate moveto $line 10 10 start
You can reshape an item so that a feature lies at a certain point: 
  $slate reshapeto $line 100 50 vertex-1
