The Slate Tutorial

This is an interactive tutorial on the Slate -- if you are running Tycho you can execute the embedded Tcl code by double-clicking on it. Otherwise, you can just read it.

Tycho contains two classes that extend the Tk canvas: the IncrCanvas class extends the canvas with complex hierarchical items, while the Slate adds symbolic item manipulation and support for user interaction. Generally, you will just use the Slate class, which is what we will be doing in this tutorial.

In this tutorial, we use the ::demo::newslate procedure, which creates a slate inside a top-level window. In your own code, you would create a slate using the slate procedure (if you are running Tycho, you need to use::tycho::slate).
