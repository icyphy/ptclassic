Tagging

Each item can have an arbitrary set of tags attached to it. For example, let's create a couple of items with the tag "fred": 

  package require slate.demo
  set slate [::demo::newslate "A tagging example"]
  set arrow [$slate create Solid \
          100 60 132 60 132 44 164 76 132 108 132 92 100 92 100 60 \
          -color green -borderwidth 4 -tags fred]
  set line [$slate create line 220 60 252 60 252 44 284 76 252 108 \
           -tags fred]

Now, almost any canvas operation will accept a tag instead of an item id. For example, to move both items: 

  $slate move fred 10 10

To find all items with a given tag: 

  $slate find withtag fred

To find the bounding box of all items with a given tag: 

  $slate bbox fred

Some operations don't make sense on multiple items, so will only work with the first item found with that tag. For example: 

  $slate coords fred
(This will return the coordinates of only one of the two items. I don't think it's possible to know which one in advance.)

We can add a tag to an item and get its tags: 

  $slate addtag mary withtag $arrow
  $slate gettags $arrow

To remove a tag from an item, go: 

  $slate dtag $arrow fred

(Check it: 

  $slate gettags $arrow
)
