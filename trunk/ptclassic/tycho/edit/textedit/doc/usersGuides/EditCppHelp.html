User's Guide to the C++ Editor

This class provides a text editor specialized for editing C++ files. It is invoked automatically when opening any file with a .cc, .h, .C, or .H extension in its name. The EditCpp class is derived from the EditC class, and therefore inherits all of its features. At the moment, it does little more than color comments.
