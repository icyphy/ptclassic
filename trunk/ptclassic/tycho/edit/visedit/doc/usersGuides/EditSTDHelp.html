User's Guide to the State Transition Diagram Editor

This class provides a text editor specialized for editing Itcl source. It is automatically invoked whenever you open any file with a ".std" extension. The EditSTD class is derived from the EditBubbleAndArc class, and therefore inherits all of its features.
