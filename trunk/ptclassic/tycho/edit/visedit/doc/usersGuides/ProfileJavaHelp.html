User's Guide to the Java Profile Viewer

This class provides a visual editor specialized for viewing the output from java -prof This profiler is derived from the EditForest class and therefore inherits all of its features. This profiler differs from the Tcl Profiler in that this profiler can only be used to view the output from java -prof, there is no way to generate a new Java profile from within this profiler.
