The Tycho Graphical Kernel

The Tycho graphical kernel package, tycho.kernel.gui, provides a powerful and flexible foundation for building graphical [incr Tcl] applications. It includes a document-view framework, and a full range of user interface widgets such as file browsers and dialogs. These classes are designed to work on concert to make building an [incr Tcl] application easier.

Documentation available:

User's guides
Guides to the user-level widgets contained in the graphical kernel. These guides are typically also available from help menus within the Tycho GUI.

Programmer's guides
Programmers guides to the classes and facilities provided by the graphical kernel. The guides include an overview of the support provides and extensive tutorial examples. If viewed in the Tycho HTML viewer, these examples can be executed by double-clicking on them.

Class documentation
Documentation for all of the graphical kernel classes, generated from the source code by the tydoc utility.
