User's Guide to the DirSearch Editor

The Directory Search editor allows the user to search multiple files for a regular expression. This editor is graphical interface to an Itcl version of the Unix egrep command. The DirSearch Editor has two windows. The first window is a file browser that allows the user to select the files to be searched and the regular expresion to be searched for. The second window is an Index Browser that displays the results of the search.

The DirSearch Editor uses the Most of the features of the file browser are exactly those of any file browser in Tycho. The differences are:

  • Hitting Return in the Directory Search Browser starts the search mechanism which brings up an Index Browser.
  • Glob characters can be used in the entry widget
  • The directory search browser does not have a button for creating a new directory.
